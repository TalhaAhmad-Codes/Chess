# Chess

**Chess** is a fully-featured board game implemented from scratch in **C#**. This project provides a complete chess experience with support for all standard rules and gameplay features.

## Features

* Standard **chess rules**: movement, check, checkmate, castling, en passant, promotion.
* Playable in **console mode** (v1) with plans for **GUI using WPF + MVVM**.
* **Save and load games** functionality.
* Designed with a clean, **modular architecture** for easy extension and maintenance.
* Prepared for future enhancements like **AI opponents** or **online multiplayer**.

## Getting Started

### Prerequisites

* **.NET 7+ SDK**
* **Visual Studio 2022+**
* Optional: **SQL Server LocalDB** (for game persistence)

### Running the Console Version

1. Clone the repository:

   ```bash
   git clone https://github.com/TalhaAhmad-Codes/Chess
   ```
2. Open the solution in **Visual Studio**.
3. Set the console project as the **startup project**.
4. Press **F5** to run.
